# Codesnap

Codesnap is a React app for taking a screenshot of the code snippet.

## Installation

Use the clone  of this repository.

```bash
npm install
```

## Usage

```npm
npm start
```
It runs the app in development mode.
Open http://localhost:3000 to view it in your browser.
